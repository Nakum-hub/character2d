# Reference Art

Turnaround reference sheets for the character. These full-body views are the
source art used to slice parts and rig the 2D live model.

## Character description

Anime-style young man:

- **Hair:** messy, layered, medium-brown with lighter highlights.
- **Top:** dark olive / brown long-sleeve crew-neck tee (sleeves slightly pushed up).
- **Bottoms:** tan/beige chinos, cuffed at the ankle, thin dark belt.
- **Shoes:** white low-top sneakers.
- **Accessory:** black wristwatch (left wrist).
- **Build:** tall, slim.
