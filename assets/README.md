# Assets

This directory holds all art assets for the **character2d** live model.

## Structure

| Folder | Purpose |
| --- | --- |
| `reference/` | Source turnaround reference art (front / back / side views). The starting point for rigging the 2D live model. |

As the project grows we expect to add:

| Folder | Purpose |
| --- | --- |
| `parts/` | Layered/sliced art (head, hair, eyes, mouth, torso, arms, legs) used to build the deformable mesh. |
| `exports/` | Final rigged model exports (e.g. Live2D `.model3.json`, runtime textures, atlases). |

> These extra folders are not created yet — they'll be added when we reach those stages.
